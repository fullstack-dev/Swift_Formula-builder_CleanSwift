## Installation

FormulaBuilderCore is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
source 'https://bitbucket.org/plumflowerteam/cocoapods-specs.git'
(ONLY need line above if you don't have)

pod "FormulaBuilderCore"
```
